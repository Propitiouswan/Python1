**Number类型**
========

**数值类型**
----
 1. 整型  Integer 
 2. 长整型  Long integer —— 只适用于Python2
 3.  布尔型  Boolean 
 4. 双精度浮点数  Double-precision floating
 5. 复数  Complex number

**操作符**
---

 1. 算数操作符

		A + B  加  
		A - B  减   
		A * B  乘   
		A / B  除   
		A % B  取余   
		A // B  除（结果取整数部分）   
		A ** B  幂   
		not A  取反   
		A > B  大于   
		A < B  小于 
		A == B  等于  
		A >= B  大于等于  
		A <= B  小于等于

 2. 比特操作符
 
		 A & B  与
		 A | B  或
		 ~A  非
		 A ^ B  异或
		 A >> B  按位右移
		 A << B  按位左移

**内置函数**
----

 1. 通用函数
 
		 cmp(A, B)  比较——只适用于Python2
		 str(A)  参数转化为字符串
		 type(A)  返回参数类型
		 bool(A)  参数转化为布尔型
		 int(A)  参数转化为整型
		 long(A)  参数转化为长整型——只适用于Python2
		 float(A)  参数转化为浮点数
		 complex(A)  参数转化为复数
 
 2. 特定函数
 
		abs(A)  取绝对值
		coerce(A, B)  类型转换——只适用于Python2
		divmod(A, B)  除模运算
		pow(A, B)  幂
		round(A)  四舍五入为整数
		hex(A)  参数转化为十六进制
		oct(A)  参数转化为八进制
		chr(A)  转化为ASCII字符
		ord(A)  chr(A)的反函数
